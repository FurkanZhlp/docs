---
title: Installation
---
Follow the steps below to install and set up **TimedWings** on your Minecraft server.

---

## 📥 1. Download the Plugin

Download the latest version of **TimedWings** from:

- [SpigotMC](https://www.spigotmc.org/resources/timedwings.00000/) *(example link)*
- [Polymart](https://polymart.org/resource/timedwings.00000/)

---

## 📁 2. Install the Plugin

1. Stop your Minecraft server (if running)
2. Place the downloaded `.jar` file into your server’s `/plugins` directory
3. Start the server once to generate configuration files

> 🔁 After the first run, configuration files will appear in `/plugins/TimedWings`

---

## ⚙️ 3. Configure the Plugin

Open the `config.yml` and adjust:

- **Flight durations**
- **Storage method** (`yaml`, `sqlite`, `mysql`, `h2`)
- **Language options**
- **Placeholder & action bar settings**

Documentation for each config option is available in the [Configuration](./configuration) page.

---

## 🧪 5. Test It!

- Give yourself permission to use TimedWings (or assign a rank)
- Use `/tw give <player> 10m` to grant flight time
- Start flying!
